OWL RAILS APP
====================

This is a Rails app which provides the main OWL API

## Installation

Requires

- Ruby (Ruby 1.9.x recommended)
- Rails

1) Copy `example.database.yml` in `rails/config/` to `database.yml` and configure.

2) Install application

    cd rails/
    bundle install
    rails server

3) Run rails server

    cd rails/
    rails server
